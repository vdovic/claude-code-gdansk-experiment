# Quick Start: How to View Church Images & Sources

## 🖼️ Viewing Church Images

### Step-by-Step Guide

**Option 1: From Timeline (Desktop)**
1. Look at the timeline view with the church lanes
2. Click on any church name in the left sidebar OR click on any event dot on the timeline
3. A **detail drawer** slides up from the bottom
4. Scroll down to find the **"Church Images"** section
5. You'll see a 2-column grid with:
   - Left: **Exterior photo** with caption
   - Right: **Interior photo** with caption

**Option 2: From Map**
1. Click the "🔍 Show Map" button to expand the map
2. Click on any church marker (colored symbols on the map)
3. The detail drawer opens with church information
4. Scroll down to see **"Church Images"** section

**Option 3: Mobile**
1. Tap on a church in the timeline or map
2. The drawer slides up (full screen on mobile)
3. Scroll down to find images
4. Tap on image to zoom if needed

## 📚 Viewing Source Links

### Where to Find "Learn More" Section

In the same detail drawer:

1. Scroll past the church images
2. Below "Most Similar Churches" section
3. Find **"Learn More"** section with links

**You'll see:**
- 📖 **Wikipedia: Church Name** → Opens Wikipedia article
- 📖 **Official Church Website** → Official church info
- 📖 **Gdańsk Tourism: Church** → Tourism/local history

Each link opens in a new tab!

## 🌍 Map Page Information

All 16 churches visible on the map:

**Map Marker Colors:**
- Color indicates church's **religion in selected year**
- Red = Catholic
- Blue = Lutheran
- Purple = Calvinist
- Gray = Secular/Unknown

**Map Marker Shapes:**
- 🟢 **Circle** = Parish church
- 🟪 **Rotated Square** = Monastic church
- 🟫 **Square** = Hospital church

**Brightness:**
- **Bright** = Selected in timeline filter
- **Dim** = Not selected in timeline filter
- Both are still clickable!

4. **Learn More:** Click source links to read Wikipedia articles while viewing images

## ❓ Troubleshooting

**Images not loading?**
- Check internet connection (images from Wikimedia Commons)
- Wait a moment for images to lazy-load
- Refresh the page

**Source links not working?**
- Check internet connection
- Try clicking again
- Links should open in new tab/window

**Can't find church images?**
- Make sure drawer is fully scrolled down
- "Church Images" section comes after the header
- Try a different church

**Church not on map?**
- Check if founding date is before selected year
- Make sure timeline filter isn't hiding it
- Try moving year slider earlier
