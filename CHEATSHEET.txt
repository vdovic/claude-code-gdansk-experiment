TROUBLESHOOTING:
────────────────
Q: Images not showing?
A: 1. Check browser console (F12)
   2. Verify server running: python -m http.server 8080
   3. Check file paths in church_images.json

Q: Script doesn't find images?
A: Check file names match: {churchId}-exterior.jpg, {churchId}-interior.jpg

Q: Need to use different file names?
A: Edit church_images.json to update URLs, then run sync script
